
# SQL Consultas Essenciais (SQL 3C + 12 Queries)

Este repositório reúne um modelo mental simples para destravar SQL (**SQL 3C**) e uma coleção de **12 consultas essenciais** para praticar com mini bases (ex.: MySQL).

## ✅ O que é o SQL 3C?
O SQL 3C ajuda você a pensar antes de escrever query:

- **Cliente:** quem pediu a informação e como vai usar
- **Consulta:** qual é a pergunta do negócio (frase testável)
- **Contexto:** de onde vêm os dados (tabelas, chaves, filtros, risco de duplicidade)

Quando você define os 3C, a query fica mais clara e objetiva.
